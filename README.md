## Тестовое задание

Для разворота нужно:
- скопировать .env.example в .env 
- указать в .env полный путь до корня проекта в свойстве DB_DATABASE

Далее выполнить команды:
- php artisan key:generate
- php artisan migrate (create database? yes)
- php artisan db:seed
